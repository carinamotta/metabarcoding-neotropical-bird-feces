# Comparing DNA Metabarcoding & morphological seed analysis of Neotropical bird feces

## Author
Carina Motta (carina.i.motta@unesp.br; carinaisabellamotta@gmail.com)

## Description
This dataset contains information on bird fecal samples collected in Corumbataí River Basin, São Paulo, Brazil between August 2022 and August 2023. Seeds found in feces samples (n = 64) were morphologically classified as morphotypes and material from the feces was sequenced using metabarcoding. Plant DNA extracted from 52 seed-containing droppings was amplified using universal plant primer for a rbcL mini-barcode. Morphologically detected seeds were cross-referenced with metabarcoding-detected genera. Assembled reads are provided here (data > processed_data > metabarcoding) but raw sequence data are available in the NCBI BioProject and in the Sequence Read Archive repository under Accession number PRJNA1203590.

## Folder Structure & Files Included
- **`analyses/`**
  - `data/`: all data used in analyses, saved as .csv from full, raw datasets (.xlsx)
  - `figures&results/`: figures and results
  - `scripts/`: script used to conduct analyses and create figures 
    - `gen_morph_script.R`

- **`data/`**
  - `raw_data/`: _Raw datasets generated by this project. Each .xlsx has a "metadata" tab to explain the column names of remaining tabs_ 
    - MottaCarina_bird_captures.xlsx: _Capture data and corresponding dropping collection. Contains three tabs: metadata ; bird_captures ; bird_spp_fam. Plot number corresponds to external dataset, "newfor_plots_metadata.csv"_
    - MottaCarina_droppings_seeds.xlsx: _Seed IDs based on morphology. Contains three tabs: metadata ; seeds ; morphotypes_
    - MottaCarina_droppings_metabarcoding.xlsx: _OTUs, number of reads, plant material identified using metabarcoding. Contains four tabs: metadata; extractions&PCR ; pools ; metabarcoding._
  - `external_data/`: _External datasets obtained from collaborators. Contains collection location metadata curated by [Dr. Pedro Brancallion's NewFor Project](https://esalqlastrop.com.br/capa.asp?p=242)_ 
    - newfor_plots_metadata.csv:
  - `processed_data/`
    - `metabarcoding/`
      - `OTUs/`
        - _Assembled OTUs of metabarcoding results; file names correspond to full dataset (data > raw_data > MottaCarina_droppings_metabarcoding.xlsx > metabarcoding)_. _Control samples are denoted as "control"._
      - `reads_per_OTU/`
        - _Number of reads per assembled OTU; file names correspond to full dataset (data > raw_data > MottaCarina_droppings_metabarcoding.xlsx > metabarcoding)_
